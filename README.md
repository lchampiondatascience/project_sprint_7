# project_sprint_7
Esta es una aplicación web interactiva para explorar el conjunto de datos de anuncios de venta de carros en Estados Unidos ('vehicles_us.csv')

El Histograma muestra la distribucion del odometro de los vehiculos anunciados.
El grafico de dispersion muestra la relacion entre el odometro y el precio de venta. 
Al hacer click en su respectivo botón se crea cada grafico.

Se usó:
Python
Pandas
Plotly
Streamlit

Este proyecto fue realizado por Laura Ubate para el Sprint 7 de Triple Ten.
